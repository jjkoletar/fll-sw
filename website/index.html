FLL Software

This is the home of the First Lego League (FLL) software. Currently the only software is for scoring and display during tournaments. This software has been written by individuals in Minnesota and has been used extensively there.

Partners known to be running the software include:

  • Minnesota
  • Illinois

If you are using the software for your tournaments, please let me know so that I can add to this list.

  • Features:
    • Open Source Software
    • Freely distributable
    • Platform Independent (written in Java)
    • Scalable - Successfully used in tournaments with 16 to 80 teams
    • Easy to use web interface
    • Allows hierarchical tournament structure to do tournament advancement from regional tournaments to state and beyond
    • Easy to configure for different tournaments with a simple XML file, no code changes required
  • Future plans:
    • Currently the install takes more time than I'd like. The next step is to have the install be 1 zip file. This release should be out late spring to early summer 2006
